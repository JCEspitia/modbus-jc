Instalación

Para instalar el proyecto, sigue estos pasos:


$ git clone https://github.com/tu-usuario/tu-proyecto.git
$ cd tu-proyecto
$ npm install

Uso

Para utilizar el proyecto, ejecuta el siguiente comando:


$ npm start
